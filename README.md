# COMP2090SEF_GroupProject
2048 Game - Python Tkinter Implementation
A classic 2048 number merge game built with python's tkinter library.
Overview:
This project recreates the iconic 2048 game: move number tiles with arrow keys to merge identical numbers, aiming to reach the 2048 tile. It uses only Python's built-in Tkinter for the GUI, with no external dependencies.

Game2048/
├── main.py          # Main game logic (runnable)
├── colors.py        # Color configuration for tiles/background
└── animation.py           # wait for update

Features:
Arrow key controls (↑/↓/←/→) for tile movement
Random tile generation (2 with 90% chance, 4 with 10%)
Win/lose detection (win at 2048, lose when no moves left)
Movement animation will be updated in APRIL

Requirements
Tkinter (built-in with Python)
How to prepare Tkinter:
mac:use terminal
#brew install python-tk

How to Run:
press ▶️ key

Key	Action
↑	Move tiles up
↓	Move tiles down
←	Move tiles left
→	Move tiles right
